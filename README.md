# PteroBackup-Lite

A lightweight Bash backup solution for Pterodactyl Panel and Wings servers. Configure automatic backups in minutes with support for multiple storage providers.

## 📋 Features

- **Easy Installation**: Interactive setup wizard
- **Multiple Storage Options**: Google Drive, Amazon S3, Mega, Local Storage
- **Flexible Backup Types**: Panel, Wings, or both
- **Automatic Scheduling**: Cron integration for regular backups
- **Backup Modes**: Keep all backups or replace previous ones
- **Detailed Logging**: All operations logged to `/var/log/pterobackup.log`
- **Colorful Terminal UI**: Progress bars and loading spinners
- **Ubuntu Support**: 22.04 and 24.04
- **Pure Bash**: No external dependencies except standard tools
